Here are all of the files you will need to install and play Action Quake2.

First off you need the full version of Quake2 installed on your pc and you need to make sure you have upgraded to version 3.20 which is the final patch for Quake2. The CD version of Quake2 is v3.05. If you have never upgraded before grab the full upgrade If you're upgrading from v3.12 or greater you can use this smaller upgrade. Both of these files include the vwep (visible weapon) patch and the full upgrade also has the latest version of CTF v1.02(capture the flag). Ive also included the vwep patch for v3.15 or greater.

Second you need the latest version of Action, it is v1.0c. grab either the exe or the zip file here. Although the exe file is recomended both have been provided for your convenience. IMPORTANT: If you are upgrading from a previous version of Action you need to remove your old action directory before you run the install. Make sure you back up your config file so you dont have to rebind everything then run the install. When prompted for the path type X:\quake2\action (X being the drive) this will create a new action directory and install the files correctly.

While you're downloading maps you need to edit your config.cfg file and bind some keys for several functions: open door, reload, drop weapon, drop item and weapon. The weapon command lets you change the mode of your current weapon i.e. machinegun: full auto to 3 round bursts. pistols: full auto to single shot. Knives: stabbing to throwing.
To bind a key open the game then use the ~ key to bring down the console then type in: bind "Key" (where "key" is the key you want to bind.) command. Example: bind o opendoor
If you're new to this and dont know how to edit your config file or dont understand why you need to edit it, visit the Newbie Bootcamp at PlanetQuake.
HOT TIP: After you have customized your config.cfg file, go to the directory and rename it autoexec.cfg. This will keep Quake2 from making changes to it or resetting it.
There are many tweeks you can make to this file that will improve the game. When you edit your config file make sure you edit the one in your action directory not the one in your baseq2 directory. We also recomend that you bind and use your keyboard for movement commands so you can use the keyboard and the mouse together for optimal maneuvering and aiming. You can easily edit your keyboard by opening your Quake 2 game and choosing options, then customize controls, this will allow you to change the keyboard settings. Some players have reported problems with the game saving the keyboard settings by editing them this way. If you have problems try this: Start an Action game and when you are in hit "escape" and go into the options setting there and try editing your keyboard, this should save the changes for your Action game.
