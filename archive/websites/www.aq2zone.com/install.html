AQ2Zone File News Jan. 6th By Buddy Lee

We have the new build of Tweakit for 3D/FX cards heres the new goodies you get with this latest release of creamy 3D goodness

NOTE: This build will report that it is 3.0.9. This is to "force" a TweakIt Update. This allows TweakIt to remove any settings from the registry that have been left over and may be causing problems. For example, the "Error 13 : Type Mismatch". The final build of TweakIt will be 3.0.8.

Added option to switch between STRING and DWORD Banshee registry settings. Once I compile a list of which cards use either STRING or DWORD settings, I will link this option directly to the card options. If you are certain that your card uses on or the other settings, please email me with your Banshee information (card, driver version, STRING or DWORD, etc) at Banshee Drivers.

Added GRXCLOCK and MEMCLOCK (along with GRAPHICSCLOCKING and MEMCLOCKING) for use with some cards.

Corrected Banshee Clock slider not updating after changing values from the dropdown box.

Additional detection code for Voodoo 2 settings added

Fully enabled GAMMA dropdown for selecting Direct3D, Glide or Both
