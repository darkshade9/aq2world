First of all get the latest patch (4.32 I think)

Open Unrealtournament.ini located in your: UnrealTournament/system folder and change this line: (in bold)

------------------------------------
[URL]
Protocol=unreal
ProtocolDescription=Unreal Protocol
Name=Player
Map=Index.unr
LocalMap=CityIntro.unr
-------------------------------------
To this line:
-------------------------------------
Protocol=unreal
ProtocolDescription=Unreal Protocol
Name=Player
Map=Index.unr
LocalMap=Entry.unr
--------------------------------------
It takes exactly 6 sec on my pentium3 450
with 196 ram and Geforce256 to run UT

This changes the UT intro map with the UT logo...
