It's a server-side command. So your server needs to have the limit set. There's nothing you, as a player/client, can do. It's all up to the server operator.

] RCON SET WEAPONS 4

(to allow 4 special weapons to be picked up, for instance)

As you might imagine...

] RCON SET ITEMS 3

Anyway, in the inetconfig.cfg file, for instance, you'd have the lines:

set weapons 4
set items 3
