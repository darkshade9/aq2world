Site Updates
SICK1 Aug 6. 1999
I've added the menu now, and the new sections will be up and running soon. The AQ:TE is looking great, especially for us clans. Can't wait for it to be ported to the 1.52 code.
Gangsters is almost dead. People don't play it so much, even thought it's great 8(. They've included a gl_kicker function (not available for download yet) which kicks people with too high gl_modulate values. Gl_modulate makes everything much brighter and on some maps, this is a great advantage, like urban2... Using high gl_modulate values are not the way action was meant to be, so please don't use anything higher than 2.5, it ruins the game!

I'm also looking forward to the new mod from aqdt, AQ:arena. This is going to be cool. I don't think there will be many servers running this mod, but I guess it'll be a great lan mod, perfecting each others skills....
