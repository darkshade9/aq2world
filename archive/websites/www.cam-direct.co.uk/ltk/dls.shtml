Downloading AQ2: LTKTBM Edition.

Along with the standard HTTP downloads you will find linked from these pages, introduced for the release of 1.7 is an awesome technology called bittorrent. From a providers POV it allows secure provision of downloads at minimal bandwidth expense to the site, from a users POV it should allow download speeds up the the maximum speed of your connection. It works by sharing the uploads between clients who already have the file, Those familiar with other peer to peer networks such as kazaa will know the benifits of these types of networks, however Bittorrent not only comes with no privacy issues, but also allows us to provide you with direct links to the files, and the assurance that what you are downloading is what you expect.

Other sites who wish to provide mirrors to ltktbm, but dont want to commit to the space or bandwidth commitmants can provide local links to the .torrent file and still provide vistors with high speed and secure downloads without infringing on other sites bandwidth.

However, as always, there are drawbacks to this technology. It relies totally on you, the user leaving the download client open after downloading so other users can recieve the file from you, so we're also, for now at least staying with reliable http mirrors as well.

EVEN IF YOU USE THE HTTP MIRRORS: If you have some spare bandwidth on your connection, you can, if you wish 'seed' the file for other users by clicking on the .torrent link for the respective file, and selecting the save location in the torrent client as the file you have downloaded.

Four types of downloads are available:

The full Installation, if you have no other files, or a standard Action Quake 2 installation, this file will setup and install AQ2: LTKTBM Edition on your machine, ready for you to play.

The updater installation, This file will install and setup the latest version on your machine (full installations 1.3b and up, version 1.2b will not be updated correctly by this file, you will need the full installation)

Additionals: This files can be used to add new features and content to a ltktbm installation

Routes: These provide single player pathfinding capabilities for maps not included in the full or update installations.
