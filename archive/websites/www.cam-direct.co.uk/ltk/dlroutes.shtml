DOWNLOADS - ROUTES

The following maps are all alpha versions and have not been tested, please use them and report any problems on the forum. Each zip contains the crtcompiler which must be run after each zip is extracted. Extract all the files to "quake2\action\terrain" note the file clist.txt is the file crt compiler uses to extract the routes.

ROUTES: - These give support for maps not included in the full installation, currently they are just converted .ltk files and have not been properly tested, If you wish to provide other users with support for these maps please read the "debugging nodefile" documentation below. Note, if you want to use these maps on the stand alone install, for the moment you will need to extract the base Q2 map textures from the ORIGONAL Quake 2 pak0.pak, use Qped on the download page for this. If you have a favourite map not included in the full install, why not get involved and redo the textures needed for it, specific instructions on how todo this will follow, but you will need a program called wally and a good image editor for this.
