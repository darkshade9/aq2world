Updated 9-13-98

You will need to pick up the Action Quake 2 mod before you can check out these demos. Action Quake 2 is one of the best mods of all time. I highly recommend getting it, unless of course you suck.

Read the readme.txt file to learn how to play the demos, if you already know how to do it, then go play the demos!
