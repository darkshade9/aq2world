for 2x, just "weapon", for 4x bind key to "weapon; weapon" you return you would need a seperate bind for each zoom "weapon; weapon; weapon" for 2x, "weapon; weapon" for 4x (just same key that zooms to 4x)
Example:
//2x zoom
bind CTRL "weapon" //zooms to 2x
bind / "weapon; weapon; weapon" //returns to normal from 2x

//for 4x zoom
bind ALT "weapon; weapon" //zooms first time, returns 2nd time

I hope this helps, it was kind of confusing, and it has an extra button that u didnt want; but i don't think there is any other way to do it.

If you put the zoom lines in an alias you can then rebind the zoom key to reset to the base level with the same key ... for example (taken from my web site alias list) ...

bind x xscp
alias zscp "weapon ; weapon ; sensitivity 1 ; bind x zscp "
alias xscp "weapon ; weapon ; sensitivity 9 ; bind x xscp "

the x can be any key you want to use for the alias and the sensitivitys can be set to whatever you like to use (you can find your base sensitivity in set in the config file if you dont have it set in your autoexec) ...

This alias is for a x4 zoom ... for a times 2 the first line would have only one "weapon" command in it and the second would have 3 "weapon" commands ... also if you use diferent aliases of this type yu will have to change the alias name so that dont conflict , anything will work but it should be easy to remeber what it is ... sa2 and sa4 for sniper alias 2 and sniper alias 4 for example ... zsa2/xsa2 ... zsa4/xsa4 ...

one last note , the x2 alias can also be used as an quick reset for sensitivity if you get caught out of wack (killed wail in another sensitivity) ...

hope this is helpful ... nb ...
