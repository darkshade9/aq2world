Any clients that connect need to get the small client pack - it's basically the normal AQ weapon sounds renamed to fit the new sound scheme and should have no effect on regular AQ servers.

damn, damn, damn.... the client pack was supposed to have the sound/weapons path in it... the first version did... grrr.... ive uploaded a fixed version to telefragged... if you have the old version (with no path names), unzip it into action/sound/weapons

here's the way the sounds map:

Mk23 - action/sound/weapons/blastf1a.wav
MP5 - action/sound/weapons/machgf1b.wav
MP5 - action/sound/weapons/machgf2b.wav
MP5 - action/sound/weapons/machgf3b.wav
MP5 - action/sound/weapons/machgf4b.wav
MP5 - action/sound/weapons/machgf5b.wav
HC - action/sound/weapons/sshotf1b.wav
SSG-3000 - action/sound/weapons/hyprbf1a.wav
M4 - action/sound/weapons/rocklf1a.wav
silencer - action/sound/weapons/rocklr1b.wav

The silencer sound should just be something that's quiet, cause it gets played in addition to the M4 sound. If you want to use different sounds, just copy to the appropriate sounds from the above list. And yes, the MP5 sound needs to be copied into all 5 files.

Note also that these files will have no effect on normal servers, just the ones running the lowlag dll.
