About Lowlag AQ2

When you fire a weapon, any weapon, the original 1.52 code actually plays TWO sounds for that shot, generating a lot of network traffic that bogs down a server in a firefight. What this modification does is use a Q2 built-in macro to play the weapon sounds.

Unfortunately for you, this requires a small modification of your own to work. Either you need to download a small client pack, or you need to rename your sounds as below. On AQ:ETE servers, if you hear the Quake 2 sounds being played, type "lowlag" in the console for instructions on how to download the lowlag sounds directly from the server itself--no additional downloading needed!

For those of you with the AQ:ETE Support Pack, you already have these sounds in the proper places and don't need to download the stand-alone lowlag client. If you have the AQ:TNG client pack then you already have lowlag sounds as well--they are included in AQ:TNG's default install!
