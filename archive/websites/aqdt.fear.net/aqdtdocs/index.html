AQDT Documentation - April 9th, 2002

Here are the docs! These are updated for April 9th, 2002. AQ:ETE 4.6b is the current OFFICIALLY RELEASED and STABLE version. Please use this instead of ANY older versions.

These docs are available for download in the new AQ:ETE Support Pack in the Files section. Once you download the AQ:ETE Support Pack and install it in your /quake2 directory, you can find these docs on your hard drive in:

/quake2/action/docs/aqdt/index.html

Also, please check out the following files in the /quake2/action/ directory:

list_aq2.cfg
list_esp.cfg
list_flags.cfg
list_rcon.cfg
list_te.cfg
match.ini
arenaserver.cfg
espserver.cfg
espserver-advanced.cfg

Also, in that directory, if you did not get the ETE Easy Support Pack, REMEMBER TO RENAME "pakN.pak" TO SOMETHING LIKE "pak9.pak"!

Click on the AQDT image above at any time to go to the main AQDT site.
