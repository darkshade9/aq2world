All files should be extracted to the "quake2" directory, NOT into
the "action" directory like the AQMD map ZIPs get extracted to.

(RELEASE version)

ete-46b-server.zip (1.1 MB)

    binaries are both included

Espionage - Support Files

ete-support-pack.zip (3.2 MB)

  • RENAME "pakN.pak" to
    something like "pak5.pak"
  • Includes 4.6b server binaries,
    HTML docs, espskins,
    and lowlag sounds
  • Does NOT include pictures from
    "How to Play" (get Doc Pack below)

  • RENAME "pakN.pak" to

    "How to Play" (get Doc Pack below)

  • If you don't want to download the
