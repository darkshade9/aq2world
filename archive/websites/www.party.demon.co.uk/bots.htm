The CGF Bot 
Much as we all love fraggin real live opponents online, playing this way can be a very expensive business. Also if you are a new player, it can be extremely disheartening to join a game and find yourself dead within seconds and totally unable to make kills of your own. Enter the Bots, computer controlled AI players that are designed to fight and behave like real people. Wanna learn how to use a new weapon? need to brush up your sniper skills? then grab yourself a mission and get practising before you take your finely honed skills online!

The CGF Bot for Action Quake 2 is a little different from most other Bots in that games are not based on an out and out deathmatch, kill em all scenario. Bot play with CGF is mission based, where you will have specific objectives to achieve before the mission is declared a success. Yeah ok these objectives can purely be to kill the other team, but assassination and bodyguard style missions are also available, and I'm sure as more missions are released new scenarios will be introduced.

Ok now you are all itching to give it a try what are you going to need? Firstly download the cgf_0_78.zip (2.3mb) main Bot file. This file contains all the Bot files required along with two missions to get you started. please note that this is still a beta programme so check back often for the latest version and or bug fixes. Once you have downloaded this file simply extract it into your "quake2/action" directory. All future mission files should also be extracted into this directory. Assuming you have a correctly installed and working version of Action Quake 2, all you now need to do is look in your "quake2/action" directory and run one of the two mission bat files, either cgf_urban_hitman.bat or cgf_urban_streets.bat, and voila!! Roberts your mothers brother you're off :-)

As time goes by and you acquire a large collection of mission files, you will probably find it easier to use the CGF Bot Launcher programme which is a handy utility that organises and displays all the missions you have installed, allowing you to quickly and easily launch missions as well as altering mission settings such as difficulty etc. To run the launcher programme you will have to download & install both Launch100.exe (772kb) & runtimes.exe (1.15mb). It is probably advisable to also download the file LaunchFix.zip (214kb) in case you run into problems using the main Launcher programme.

The Missions
The missions are what make the CGF Bots such fun to play. Simply download one of the missions listed and extract the zip file into your "quake2/action" directory. Then run the freshly extracted bat file and your away. For instance should you download and extract the mission snipers, you will then run the file cgf_urban_snipers.bat. please note the following:

Missions marked with a tick worked 1st time, with no problems straight from extraction on "MY" PC.

Missions marked with a cross failed to run or simply locked up my PC when run after extraction.

However, just because a mission failed to run on my computer does not mean it wont run on yours. I know for instance that the Leon mission does work on other PC's and it does strike me as odd that none of the City map based missions are running on mine. There could possibly be a problem with my version of the City map, or it has just struck me that I am running the very latest, just released version 0_78 of the Bot. I will try all the missions again on build 0_77 just to be certain.
