Custom Gamespy Tabs  
by Shinobi    

  

I will explain how you can create custom tabs for action quake which will only display either teamplay servers or deathmatch servers. It's a fairly simple process which shouldn't take much time.

Start up gamespy and at the top menu bar go to view->custom tab->add. Name this tab action quake deathmatch or something similar. Next, switch over to the "Games For Tab" tab. Uncheck every box except for quake2. After you've done that, switch back to the "General" tab. Check the box which reads, "Run a custom game for servers in this tab." For the game pulldown menu, either select action or type it in manually.

Select the filtering tab and hit the button that says, "Reveal advanced options". Click on the add button. Use the pull down menu to select the option that says rule. Another box should appear. In this box select the option that says gamedir. At the text compare menu, select does, equal, and in the box type in action. Click on the ok button. Now we have to repeat the process to add the deathmatch filter. Hit the button that says, "Reveal advanced options". Click on the add button. Use the pull down menu to select the option that says rule. Another box should appear. In this box select the option that says teamplay. At the text compare menu, select does, equal, and in the box type in 0. Click on the ok button.

Here we go again. Hit the button that says, "Reveal advanced options". Click on the add button. Use the pull down menu to select the option that says rule. Another box should appear. In this box select the option that says deathmatch. At the text compare menu, select does, equal, and in the box type in 1. Hit the ok button twice. Congratulations, you've just created a deathmatch only tab.
