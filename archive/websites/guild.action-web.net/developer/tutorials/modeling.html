The very first thing you'll need to sort out is an idea for a model. If you have reasonable 2D skills i'd recommend you get your idea onto paper first. In general i've found that models which start from a drawing will end up better than any make it up as you go along one, even though they don't always end up anything like your drawing.

This tutorial is going to be based on how i created the chicken model. (Which itself was my first attempt at a Quake 2 Model)

As this tutorial may require numerous pictures i've split the process down into sections so you can quickly find whatever you need.
