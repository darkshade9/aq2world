���Wednesday, January 05, 2000 - 9:27 AM EST Hammer - How to get fog in AQ, by kramed
kramed has discovered how to get fog to work in AQ2. It's a client side thing, so you can try it if you want!! You need Opengl 1.1 drivers for your card

This morning by fluke (I was switching the shotgun model) I found out that you can get the fog from CGF and LTK working in regular multiplayer games.

Fog: http://www.botepidemic.com/aid/cgf/shots/riot_fog1.jpg http://www.botepidemic.com/aid/cgf/shots/monastery1.jpg

Default opengl users can skip next 2 paragraphs.

3dfx users can get this working too. Just download the mini gl used in GlQuake (opengl.dll), put it in your quake2 directory and change from 3dfx gl to default gl. On my P200 I got an extra 1.5 fps. I use default opengl all the time now. There are no visible diferences either. Get the latest GLQuake opengl.dll: http://tux.telefragged.com/file.pl?filename=opengl32.dll&dir=telefragged/actionworld/kram/

After you put you stick that in your quake2 folder go into regular quake2 and change the video setting. Then when you go into action it will have the right settings. (BTW, when you play CGF you gotta rename the opengl.dll to something different or CGF will crash).

Now goto http://www.botepidemic.com/aid/cgf and download the latest client/server pack. You only need the folder /action/cgf/fog if you dont want the whole thing. Then goto http://www.planetquake.com/maxtron/ltk/ and download the latest LTK dll. Stick it in you quake2 folder (backing up 1.52 and CGF).

Edit /action/cgf/fog to your liking (my default one is: default 0.09 1 0.1 0.1 0.1 200 2200), set fog 1 in your config, start up AQ2 with the ltk dll, start a game, the bots will join, disconnect and join an internet game. You will have fog. yay. Once you start a map with the bots and the fog every game after will have the fog. Pretty nifty.

I agree, that's pretty nifty.
