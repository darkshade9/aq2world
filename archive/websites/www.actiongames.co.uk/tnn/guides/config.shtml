Action Quake 2 Configuration

Quake 2 is arguably the most customisable game ever made, you can change almost everything about the way you game behaves and the way you control it. While it is possible to change some things through the menus and most things through the console the best way to do it is undoubtedly by creating a permanent personal config file.
