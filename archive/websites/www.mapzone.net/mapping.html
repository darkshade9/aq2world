MapZone Mapping Resources
Map Editors

QERadiant - QERadiant is what I am personally familiar with and will only use for making Quake2 maps. There are many tutorials for this, and I've never had a problem with it.

BSP - I've only used this editor once, but it is as widely used as QERadiant, plus has many different features.

Qoole - Qoole is an editor widely used by beginning mappers, because of it's ease of use and simplicity.

Texture Editors

Wally - Wally is the texture editor of choice by many, if not all mappers.

*.pak Editors

PakExplorer - This *.pak editor is known for it's simplicity and makes it easy to edit a *.pak file , but has limitations.

QPed II - QPed II is a more complex *.pak editor that lets the user make heavier modifications in the *.pak file.

Prefabs

FX - I highly recommend this site for advanced prefabs. It has many examples with tutorials that will help the mapper learn how a certain prefab works, and is a great tutor that spices any map up.

Other

Rust - This site is the ultimate mappers' database. Along with it's explanations and tutorials, it provides many links to the latest mapping discoveries.

GenSurf - Having trouble creating a nice outdoor environment? This is a highly recommended tool to help create that surface you gave up on making.

.txt Gen - Every map must come equipped with an information loaded *.txt file! This will help you make that task become very organized and look professional.

Radiosityland - Home of ArghRad, the compiling utility that with, one of it's many abilities and functionalities, can add lightning to your map that replicates the sun.
