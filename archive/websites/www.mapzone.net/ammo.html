Reaction Quake 3 - Ammo

Ammo placement in Reaction Quake 3 Teamplay is non-existant. Ammo placement in RQ3 Deathmatch is done by simply using the Quake 3 ammo entities and checking this conversion list:

ammo_bullets = H&K Mk23 Pistol clip
ammo_shells = M4 Assault Rifle clip
ammo_slugs = SSG3000 Sniper Rifle bullets
ammo_cells = MP5/10 Submachinegun clip
ammo_rockets = Benelli M3 Shotgun and Handcannon shells
ammo_grenades = Nothing just yet
ammo_bfg = Nothng yet
ammo_lightning = Nothing yet
weapon_gauntlet = Combat Knife
