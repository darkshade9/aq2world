Reaction Quake 3 - Ladders

To make your brushes climbable in Reaction Quake 3, a shader must be applied to the brush you'd like climbable. To get this to work on a simple ladder that only has a texture applied to it, you'd need to make a shader file or add on to an existing one if you're more experienced with shaders. For this example, our ladder is covered with the gothic_ceiling/woodceiling1a texture, but we'll need to make a shader for this. Create a text file called rq3laddy, and paste this text from this text file into it. Now, save it and rename rq3laddy.txt to rq3laddy.shader and place that into /quake3path/baseq3/scripts then open up the shaderlist.txt file in that folder, scroll down to the last line and add rq3laddy. All done, now save the updated shaderlist.txt file, load up your map editor and apply gothic_ceiling/woodceilingladder to the simple ladder you made. It will now have the gothic_ceiling/woodceiling1a texture applied to it, and it'll be climbable.

Another way to make an area climbable is to make a brush and cover it with the Ladder Shader provided below, which uses a special texture to make it easier to see where the ladder brush is. Just save the ladder.shader file below into /quake3path/baseq3/scripts and add the word ladder to the last line of the shaderlist.txt in that folder, save that and you're ready to make an invisible ladder(Save the ladderclip.tga file to /quake3path/baseq3/textures/common - Then when you load up Radiant just load the /textures/common directory and you should see "ladder" there). The advantage to this is if you only want a part of the ladder climbable, just shape the brush to however you want the player to be able to climb upon, and put the ladder shader on it. In Radiant there should be a translucent brush covered with the word "ladder", now the player will be able to climb on that, so place it against a wall or ladder to use it effectively.
