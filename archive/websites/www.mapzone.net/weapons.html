Reaction Quake 3 - Weapons

Weapon placement in Reaction Quake 3 Teamplay is non-existant, the player gets to choose their weapon before they can start playing. Weapon placement in RQ3 Deathmatch is done by simply using the Quake 3 weapon entities and checking this conversion list or use the RQ3 entity names for them listed below:

Reaction Quake 3 Weapon Entities

weapon_knife = Combat Knife
weapon_pistol = H&K Mk23 Pistol
weapon_m3 = Benelli M3 Shotgun
weapon_mp5 = MP5/10 Submachinegun
weapon_handcannon = Handcannon
weapon_m4 = M4 Assault Rifle
weapon_ssg3000 = SSG3000 Sniper Rifle

Quake 3 Replacement Weapon Entities

weapon_gauntlet = weapon_knife = Combat Knife
weapon_grenadelauncher = weapon_pistol = H&K Mk23 Pistol
weapon_shotgun = weapon_m3 = Benelli M3 Shotgun
weapon_machingun = weapon_mp5 = MP5/10 Submachinegun
weapon_rocketlauncher = weapon_handcannon = Handcannon
weapon_bfg = weapon_m4 = M4 Assault Rifle
weapon_railgun = weapon_ssg3000 = SSG3000 Sniper Rifle
