Reaction Quake 3 - Breakable Glass

To get breakable glass to work in Reaction Quake 3, you must put the definition file found below into your Q3Radiant (in /quake3path/tools) or GTKRadiant folder(in quake3path/extras), make sure to back up your existing entities.def file in the appropriate directory. The entity defination file below includes the func_rotating_door and func_breakable entities.

All that has to be done for this to work is to simply select the brush that you want to be your glass, and make it a func_breakable entity, set its variables as you would any other entity and you're all done!

Note: If you have a special entities definition file and know how to add an entity entry to the file, then simply copy and paste this:

//=============================================================================
/*QUAKED func_breakable (0 0 0) ? Breakable object entity that explodes when damaged.
-------- KEYS --------
health : determines the strength of the glass (default 5).
-------- NOTES -------- This entity was made for breakable glass. To correctly use this entity, cover the brush with a glass shader and determine a health for it. */
//=============================================================================
