PingTool

PingTool can be downloaded at http://www.pingtool.com.To use the following server lists, Add New Server Source (in Masters), put the filename in Description and Server Source...., make the Server Source Type a Local File (or URL).

TeamPlay List URL: http://action.telefragged.com/pingteam.txt
DeathMatch List URL: http://action.telefragged.com/pingdm.txt

Unzip DeathMatch List , TeamPlay List , into the PingTool directory.

GameSpy

GameSpy can be downloaded at http://www.gamespy.com.To use the following server list, right click on your master server list under Quake2, Add Server List..., select a Name, Browse for file (or URL) and select server list. The new server list will appear at the bottom. right click on the new list and select Update.

TeamPlay List URL: http://action.telefragged.com/gsteam.txt
DeathMatch List URL: http://action.telefragged.com/gsdeath.txt

Unzip DeathMatch List , TeamPlay List , into the GameSpy directory.
