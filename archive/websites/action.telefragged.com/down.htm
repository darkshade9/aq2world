Downloads

Before you download Action Quake2, you may want to read about it in our About Action Quake2 section.

Action Quake2 is multiplayer only.

Step one:
Prepare your computer for Action Quake2. First of all, make sure you have Quake2 installed properly, and that you have updated Quake2 to the latest version, 3.20. You may need to update your Quake2 version (not your Action Quake2). If you have never upgraded Quake2 or it is earlier than 3.12, you can grab the full upgrade. If you have version 3.12-3.15, you can grab a smaller update. If the servers are running a newer version of Quake2 than what is here then you will need to upgrade to whatever version they are running. Quake2 insists that servers be the same version as the people who play on them. If you have never played Quake2 multiplayer before, you are in for a treat! However, jumping headfirst into a mod may be more difficult than you expected. It shouldn't be, but you have been warned! =P

Step two:
Download the latest Action Quake2 client version, v1.0c. Choose either the actn1Ccl.exe installer (8.8 megs) or an ol' fashioned zip actn1Ccl.zip (8.8 megs). If you choose to download the .zip (not recommended, but might be needed for some), realize that you need a zipper that supports long file names. Try WinZIP. While you are waiting for it to download, check out step three.

You will also need to download and install these two files that fix small problems with vwep:
1cCLfix.exe
vwepfix.exe
As always, the EXE files are recommended, but here are the zip version in case you need them for some reason:
1cCLfix.zip
vwepfix.zip

You will also need to download the latest model/skin packs. You can get these at the Actor's Guild in the Downloads section.

If you want to run a server, check out our server files page for the very latest server binaries.

MAPS!!! Of course you need maps. That is where the map depot comes in handy. Click Here to get all the maps you could need.

Step three:
While you are downloading, you should read the docs! You should also set up an Action Quake2 config file, if you have never done so before. Instructions on config files can be found at The Newbie Bootcamp (it was temporarily down last time I checked) Also check out the configexamples.zip included in the docs.zip! You will want to make one specific to your tastes, and with the new Action Quake2 keys bound in it.

If you know how to make a config file, or bind keys, here are the keys you HAVE to bind (you should still check out the configexamples.zip for a ton of cool examples of stuff like sniper aliases):

  • opendoor - It opens doors for you.
  • bandage - Bandages your wounds. Your gun goes down for 6 seconds. After 3 seconds of bandaging, you will stop bleeding.
  • reload - Discharges current clip and loads a new one.
  • drop item - Drops your current special item, if you have one.
  • drop weapon - Drops your unique weapon, if you have one.
  • weapon - Changes mode of your current weapon, if applicable. For the sniper rifle, this counts as 'zoom'. For the pistol, it switches between semiautomatic and automatic. For the knife, it switches between slashing and throwing.

You also may want to bind keys to use specific weapons:

Here are the equivalents from Quake2:

Button # :

  • 1. 'use blaster' is the same as 'use Mark 23 Pistol'
  • 2. 'use shotgun' is the same as 'use M3 Super90 Assault Shotgun'
  • 3. 'use sshotgun' is the same as 'use MP5/10 Submachinegun'
  • 4. 'use machinegun' is the same as 'use Handcannon'
  • 5. 'use chaingun' is the same as 'use Sniper Rifle'
  • 6. 'use grenade launcher' is the same as 'use M4 Assault Rifle'
  • 9. 'use railgun' is the same as 'use A 2nd Pistol' (akimbo pistols)
  • 10. 'use BFG10k' is the same as 'use Combat Knife'

Inotherwords, you don't have to rebind anything for the weapons if you don't want to, just remember the equivalences.

Step four:
Hopefully you have finished downloading it by now. Before you install, make sure that you have cleaned out your old Action Quake2 directory, if you had one before. Delete it, but be sure to back up any personal files (like config files for example). Now, if you downloaded the installer, simply run it. It will ask for your quake2\action directory. For example, if Quake2 is installed to your X: drive, type X:\Quake2\action in the prompt (that's the default). If you downloaded the zipfile, unzip it, into your X:\Quake2\action directory. The subdirectory information is stored, and it will be properly set up.

Step five:
Find a server! This is a multiplayer only mod, so grab this GameSpy tab, and fire up GameSpy. If you don't know what GameSpy is, check it out!

Have a blast! Empty a clip into your archenemy for me!
