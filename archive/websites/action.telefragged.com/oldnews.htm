      • External .ini Map rotation. w000! Just finished this, it works. I feel all warm inside. Big thanks to Geist who wrote the tutorial on QDevels that helped me thru this. I used snippets of his code too, but the implementation is quite different. The way it works is simple. You put an action.ini file in your action directory, if you run a server. If you set "actionmaps 1", and the file is correctly set up, it will rotate sequentially between the maps listed in the .ini file. On the client side, when you join a server, if they are running 'actionmaps 1', it will list the maps that the server is rotating in the Message of the Day, so you can decide then whether to stay or leave, based on the server's map selection.

      Monday - July 13, 1998 - Cail (12:10 AM CST)
      I figured since I'll be gone, I'll post a little rant about balance, the direction of the mod, etc. etc. We've been browsing the message board, reading mails, and playtesting ideas, and we have a plan...
      I'll just kinda list things: (warning, this is LONG)
      First things first: this mod's focus is Action. Not realism. Realism just happens to be one of the things you find in Action movies: to an extent. Now if you've ever seen an Action movie, there is something inherintly unrealistic about them. One person does not kill 300 in a row with minor wounds. Same with quake2. If someone could go from 0-27 mph in .1 seconds, they'd probably pulp. For a quick primer on the issue of quake2 un-realism, check out sCary's editorial. We're gonna tweak the acceleration, but there are still many things that will be unrealistic about AQ2 (try respawning =P ).
      In short: fun comes first.. now on to the show:
      The pistol has become too powerful, in relation to the other weapons. Now don't worry, we're not gonna mess it up. All we plan to do with the pistol is put a range modifier: ie, if they are too far away, it does less damage. This would be scaleable, so its best to be close, but even at a max range it will do no less than half damage (maybe more). This is to promote Action fights, and make it more difficult to snipe with the pistol (absurd and boring). This will also make the Sniper Rifle more useful.
      The unique weapons are less powerful than the pistol, except for certain combos, or perhaps the shotgun, they are simply not as good. This will change, but only to a point.. We want to MAKE you want those unique weapons, and fear death, and losing them. Here's how we will change them:
      The shotgun is great the way it is (perhaps a tad more damage, as well as bug fixes (bandaging), etc. )
      The mp5: we are going to add different fire modes (single shot, 3rdburst), as well as make it simply better. Certainly better accuracy, and maybe a bit more punch too.
      The handcannon: we hope to replace the model, but if you wanted to know, its a sawed-off. It sucks right now, because the spread is a bit too high. We have many plans to try and fix it (before it was TOO powerful, and we overcorrected). Don't worry about it, it will either turn out well, or we are also thinking of making it go away.
      The sniper rifle: we WILL fix it for one last time, making it 100% accurate. We will also fix the bug where you can't do anything while zoomed out. Finally, we should be able to add a 'hip shot' that is innacurate, but you can shoot while zoomed out. ("sniper" mode)..
      A Colt M4 (assault rifle): we are going to add Gooseman's excellent model to the lineup. We will find a way to balance it, but yes it will be a damed good weapon, and the person who uses it will want to stay alive and keep it. We will either recompile the maps with the new weapon placement, or perhaps replace the handcannon with the M4 and find some way to do ammo for it. We'll see.
      Finally, Gooseman is working on Akimbo pistols (dual guns!). Hopefully, it will be in the next release, in addition to his Colt M4...
      Hmm... i think that's about it.. there is some more stuff we will be doing, but I think this ought to satiate some of you.
