Contribute Maps

If you are done with your map, and certain that it is the final version and worthy of Action Quake2, go to the Action Quake2 Map Depot to upload it!

Action Quake2 relies mostly on external people to create maps for the mod. This way, we can get many more maps then most mods have, and have more free time to work on the mod itself.

If you have made maps before, then this information is for you. If not, I suggest going to a map tutorial site like Rust, and grabbing a map editor (I suggest Qoole).

Now, on to the information:

Style

We are looking for maps with an action movie feel, a warehouse, a dance hall, a hospitol, a mall, etc. (try to be original) Scenes from certain movies come to mind, try and replicate those... If you make a quality map then it will be put up on the Map Depot and our ftp site. Except for certain entities such as the weapons and ammo, there is nothing particular to Action Quake2 you need to learn. You don't have to do anything special to make doors 'openable', other than possibly make them 'toggle'. If you, for example, don't know how to do this, try a map tutorial site like Rust.

Action Quake2 Entities

Instead of using .ent files, we've decided that using Quake2 Equivilents is the way to go. Whereever you want a certain Action Quake2 weapon or ammo, just place the Q2 Equivilent from the list below:

Shotgun = M3 Super90 AssaultShotgun
Machinegun = MP5/10 SubMachinegun
Rocketlauncher = Handcannon
Chaingun = Sniper Rifle
Grenades = M26 Fragmentation Grenade
BFG10K = M4 Assault Rifle

Here is the ammo list:
Box o' bullets = Pistol clip
Box o' shells = 7 pack of shells
Box o' rockets = Machinegun Magazine
Box o' slugs (railgun) = Sniper ammo
Box o' cells = M4 ammo

Knives can't be placed. You start with one and the only way to get more is by killing people

That's it! You need not worry about items such as the kevlar vest, they spawn automatically in respawn sites like in CTF.

For doors to work like real doors, make them rotating_door, toggle, and wait -1 so they will stay open/closed.
