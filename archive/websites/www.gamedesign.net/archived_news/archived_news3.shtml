  • Daikatana Mapping Tidbit
    • John Romero updated his .plan with this little tidbit of Daikatana mapping info.

      Also, one of the cool features of Daikatana Deathmatch is that, depending on the number of maxplayers your server is set for, the map can open up new areas or keep them closed off. If you want your DM map to be scalable for 1-4 players, you just put in a special wall that blocks off access to the other areas if maxplayers = 4. Likewise for the other maxplayer levels (8, 16 and 32). This allows DM mappers to create one single map that is good for 2 players to 32 players! You'll see this feature in our Daikatana Deathmatch Demo...

  • Winbspc v1.0 released
    • That cool program Winbspc that can decompile .BSP files into .MAP files has been updated to version 1.0. You can find out more about the new version over on gladiator bot homepage or you can just download it from the Tools and Utilities section of the Rust file server. Here is the list of changes:

      - Fixed multithreading.
      - Added number of threads selection combo box to the conversion settings dialog.
      - Added Q1 & HL best match brush side texture placement.
      - Added Q1 & HL exact brush side texture placement.
      - Added Q1 & HL brush merging.
      - Added 'best match texturing' option to the conversion settings dialog.
      - Added 'no brush merging' option to the conversion settings dialog.
      - Added Extract progress dialog with cancel option.
      - Added overwrite prompt before extracting.
      - Added script option and command-line script.


    This is definitly a cool program that you should check out.

  • Pursuit of Perfection Updates
    • QOOLE:Pursuit of Perfection has been going nuts today with updates.

    First up is some new texture list groups for Quake2 that can be downloaded. For the curious, the texture list groups are lists that enable the textures to be split up by predefined categories, like Floors/Walls/Metal/Rock. Then you can view these specified groups from within QOOLE. This is definitly one of the best new features I've seen in an editor in a long long time. Very cool.

    Second up is a cool ChangeLevel/Landmarks tutorial written by Phil Daniels. This of course being for Half-Life.

    They have also posted a HUGE list of what has changed in the newest QOOLE 99 beta, and what to look forward to in the future. Did you know that the new version has limited Sin support and Heretic 2 support? That's not all that's new, so head on over to Pursuit of Perfection and get all the details.
