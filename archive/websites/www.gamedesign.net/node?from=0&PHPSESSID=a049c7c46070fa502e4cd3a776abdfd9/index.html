Quake 4 1.1 Beta Update Available!

Submitted by Method on Wed, 2006-02-22 19:59.
id and Raven Software have released Quake 4 1.1 Beta Update, which changes and fixes quiet a lot. For the list of all changes, check out the readme file. Here's some word from the big guys themselves:

"Once we are satisfied the 1.1 BETA is completely solid, we will release the FINAL 1.1 update. This will be a required update and will be available through the “Update” option in the game. With that release, we’ll also be including 4 all-new multiplayer maps in addition to the QUAKEMAS map pack (3 maps) that was released in December."

Grab the patch here. For more mirros, visit id Software site.

VIA Q4 Tourney Mappacks released!

Submitted by Method on Thu, 2006-02-02 23:24.
VIA Q4 Level Design contest mappacks has been released.

"The maps are available to download individually, or otherwise as a small map pack format (containing the top 3 entries only), as well as a large map pack that contains all of the maps that made the original judging shortlist (12 maps total), all of which are well worth checking out. There are a few maps within that might actually suit small FFA or small Team DM matches and a couple also that might one day serve as excellent rocket arena matches. Unzip the contents of the map packs into your q4base directory and each map will appear in your multiplayer menu. We recommend reading the txt readme's that are associated with each map. "
