env_laser

This entity is used to create lasers effects.

Lasers are a little less complicated to setup than an env_beam, as they only require a minimum of 2 entities to work. You need to have an env_laser and either an info_target or a func_train. The env_laser defines the starting point for the laser, so you can't move the starting point of the env_laser around.

For an env_laser to work, you need to have at least 2 entities, an env_beam and an entity that acts as a target (point in the map) for the laser. You can use an info_target for the end of the laser if you want a laser that stays in place. If you want a laser that will move around, like if you want it to move in a circle on a door to simulate cutting a hole in it, you will need to use a func_train for the end of the laser.

To get a straight laser set the "noiseamptitude" to "0" and for a laser that jumps all around like lightning set it to anything higher than "0". The higher you go the more intense the laser will jump around.

If you want a laser that doesn't move, you will need to place an info_target entity where you want the laser to stop and give it a "targetname". Now, place your env_laser and make it's "LaserTarget" field point the same as the "targetname" of the info_target.

If you want your laser to move, you will need to setup a func_train entity for the end of the laser. This func_train MUST contain an origin brush, which will be used as the point where the laser will end. Set the "LaserTarget" field to the targetname of the func_train that will be used for the end of the laser. Once that's done, you can setup the path for the func_train, which is described here.

Valid Keys/Values and Flags
    keyvalues
    targetname - name
    LaserTarget - end point of laser beam
    renderfx - render fx (0 - 16)
      0 = normal
      1 = slow pulse
      2 = fast pulse
      3 = slow wide pulse
      4 = fast wide pulse
      5 = slow fade-away
      6 = fast fade-away
      7 = slow become-solid
      8 = fast become-solid
      9 = slow strobe
      10 = fast strobe
      11 = faster strobe
      12 = slow flicker
      13 = fast flicker
      14 = constant glow
      15 = distort
      16 = hologram (distort and fade)

    renderamt - brightness (1 - 255)
    rendercolor - beam color (R G B)
    width - width of beam (in pixels*0.1, 0 - 255)
    noiseamplitude - amount of noise (0 - 255)
    texture - name of sprite to use for beam (default sprites/laserbeam.spr)
    EndSprite - end sprite (default sprites/laserbeam.spr)
    texturescroll - how fast the texture should scroll across the beam (0 = fixed texture)
    framestart - sprite frame to start on.
    select textures on different beams from the same sprite
    damage - damage per second

    flags
      start on (1)
      toggle (2)
      random strike (4)
      ring (8)
      start sparks (16)
      end sparks (32)
      decal end (64)
      shade start (128)
      shade end (256)
