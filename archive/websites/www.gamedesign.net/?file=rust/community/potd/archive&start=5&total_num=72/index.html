AlienVsPredator a 3dsMax Tutorial.
[ 2001-12-11 | sent in by sunnebjer_johan ]

This is going to be one of the best tuts ever. -Using lots of tools and many importent modifiers. Going trough different snap options and ofcourse, some mapping tech. In this tut we are gonna use the inHouse Import/exporters, drag and drop textures from asset-manager without having to convert the textures first. This is a inverted room tech, and can be used for games like Unreal and Descent etc. I have worked with the standard UI as well so you shouldnt have any troubles with following the tut either. -I will incl, a def.max file with some needed start object, and a start shot showing witch my favo game modifiers are, and explain how to short-cut them as buttons. There is five rooms, refered as modules. But every module needs to be a single mesh, so we going to weld vertecies also. Good to know if you are planning to be a modeller to.... Need a few more days for the docs, and then i think Jabby needs some time to make this availible for the public. Iam gonna do this in two step. In this tut we are not going to put the level in the game, thats going to be the next tut, and how to control vis etc. Johan Sunnebjer alias djmj
