Lens Flares - Joel Caesar


First you need the texture set, GenFX, so in the browser, hit Load, and then load the GenFX.utx.

Next you need a light. I've noticed in the Unreal maps, that the editors used two lights, one as a light, the other as the flare. In my experiments, I've only used a single light (and a spotlight at that ;) At this point, it doesn't really matter what type of light the light is, though I suspect it needs to be a "steady" light, not one that goes on and off. More playing with UnrealEd will give the answer to that.

Bring up the lights property window, and set your light settings as you see fit. Next, make sure the Textures -> GenFX is chosen in the browser. At the top of the list you will see 5 lens flare graphics. Find a flare graphic--one appropriate to your room's ambiance. Click once on the texture to select it.

In the light properties, go to Display -> Skin and click next to it to start editing. You will see a string gadget, along with a "clear", "use", and "..." (ellipses) button. Click on the "use" button. The editing fields should now disappear, and the texture name should be listed next to Display -> Skin. In my case, it looks like:

Texture'GenFX.LensFlar.5'

Finally, go to Lighting -> bCorona (default False) and set it to "True".

Compile your map and check it out. At this point, you will probably find the size of the flare (depending on the texture you chose) to be WAY to big. Let's remedy that.

Go to Display -> DrawScale (which defaults to 1.000000) in the Light properties window, and change it to 0.3 to 0.45 or so.

Under the light's properties, in Lighting, there are 2 Fields to take note of. One is bCorona, the other, bLensFlare. These are both normally set to False. The field bCorona must be set to "True" or you won't see the flare. I've seen no noticeable changes with Lighting -> bLensFlare set to either True or False, so I left it at its default, False.

Compile.

A final note, the DrawScale appears only to affect the rending of the Skin (or LensFlare texture) and not the size or other attributes of the light. Also, if you want to create the same light somewhere else, do the copy and paste thang. If you need to change a bunch of lights to be the same, CTRL-Click all of them, and then bring up the Lighting properties window (right click) to edit them all at the same time.
