Q: How do I have my map appear in the in game map browser?

A: You need to make a .arena file that would look like this:
{ map "mapname"
longname "mapname"
bots "names"
fraglimit #
type "gametypes"
}
Place this file in the (quake3)\baseq3\scripts directory.
Example: bobdm1.arena
{
map "bobdm1"
longname "The Hate"
bots "lucy sarge"
fraglimit 30
type "FFA tourney team"
}
