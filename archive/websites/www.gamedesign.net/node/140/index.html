How to create character's deathtarget?

Home » Kingpin » Tutorials
Submitted by Kingpin_Xatrix on Fri, 2004-02-27 23:56.

DEATHTARGET

by Method

Deathtarget used with characters to trigger other entities.

Example use: select a character and open Entity inspector.

Key deathtarget
Value startmovie

Value is the targetname of the entity, which will activate after
you kill character.

You can enter any targetname for the entity.
