Steam: sv_downloadurl
Submitted by mazemaster on Monday, March 1, 2004 - 06:29


HL mappers rejoyce! Steam has implemented a server cvar named sv_downloadurl to allow players to download maps from a http server rather than the gameserver. By allowing fast http downloads for custom content, servers will be more likely to support excellent custom maps that are rarely played due to filesize.

More information is available in this thread in the Steam forums.

This cvar could breathe more life into custom HL engine maps, so admins go forth and use sv_downloadurl!
