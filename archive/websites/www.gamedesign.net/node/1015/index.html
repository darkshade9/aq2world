Source Utilities Roundup

Some talented people have produced some great utilities for Source modding. Have a look and see what makes your life easier.

I am not here to rate any of these or tell you to use one over another, just to make these utilities known to you in one easy spot. Note that Bit Torrent links are provided for your convenience and to preserve the bandwidth of the authors. Note also that tools made for HL1 that may still have a useful function in Source modding have not been included (IE: programs that export to .map, .wad or .rmf files).

All tools remain the property of their authors, and while they are available for free, you should take the time to thank the author for his tool should you use it.

Download All Tools
Download all the tools presented here in one bit torrent.
Download BitTorrent


Texturing:

Windows XP/2000 VTF Shell Extensions
By Neil Jedrzejewski
This is a shell extension for Windows XP/2000 that allows you view VTF files when using the thumbnail folder view and it lets you add custom columns in "details" folder view that lets you see everything from a VTF's dimensions to alpha. Very handy.
Homepage : Download HTTP : Download BitTorrent

VTFTool 0.5b
By Neil Jedrzejewski
This program provides a GUI alternative to the Valve command line texture utilities. It allows you create VTF files, and change a VTF back to a TGA, export mip maps, and many other texture related features.
Homepage : Download HTTP : Download BitTorrent


VTF Explorer
By Vyachelslav Djura
This utility lets you explore the steam cache files (.gcf) and view textures and their different components. It also lets you extract files from the .gcf file.
Homepage : Download BitTorrent

Auto Seamer
By Ryan Gregg
A handy little utility that has your choice of three algorithms that make your textures seamless. Fine which one works best for your particular texture, and you have just shaved a load of time off your texture creation.
Homepage : Download HTTP : Download BitTorrent

TEXgen v1.0
By James Freedman
A GUI front end for vtex that can also automatically create your VMT files for you was well.
Homepage : Download HTTP : Download BitTorrent


Mapping:

Batch Compiler
By Ryan Gregg
Compile your maps separate from hammer and save some processing cycles. Includes many specifications, which tells the batch compiler which command line options to make available. Note: requires the .NET Framework.
Homepage : Download HTTP : Download BitTorrent

RESMaker
By [FBNOps]OtakuCODE
Creates the RES file needed by a server to ensure the downloading of all resources needed by a map.
Homepage : Download BitTorrent


Modeling

3DSMax 6/7 VTF Texture Plug-in 0.2b
By Neil Jedrzejewski
Plugin for 3D Sutdio Max that lets you use VTF files as textures.
Homepage : Download HTTP : Download BitTorrent

Source MDL Texture Info 1.0
By Neil Jedrzejewski
Gives information about the textures(skins) a model uses. Extremely useful when you can not figure out why your model is still showing the pink and black checkerboard texture.
Homepage : Download HTTP : Download BitTorrent

GUI StudioMDL Source 0.1b
By Neil Jedrzejewski
Extremely handy GUI for compiling models with studiomdl. Saves a lot of typing in an archaic command prompt.
Homepage : Download HTTP : Download BitTorrent

SoftImage XSI 3.x Exporter Plug-in for Milkshape 3D
By Neil Jedrzejewski
Export meshes from Milkshape 3D so that you can edit them in XSI. Brings HL2 modeling to Milkshape.
Homepage : Download HTTP : Download BitTorrent


Misc Tools

GFCScape
By Ryan Gregg
Following the tradition of the great PAKScape of old, GFCScape is a nice simple tool for navigating and extracting files from a steam cache file. Note: requires the .NET Framework.
Homepage : Download HTTP : Download BitTorrent

Rufio's SDK GUI v1.1
By Russ Farr
A GUI for both VTEX and studiomdl. More features are planned in the future, so keep on eye on it.
Homepage : Download HTTP : Download BitTorrent

Microsoft Visual C++ Toolkit 2003
By Microsoft
The compilers/linkers, C++ runtime libraries, and framework for MSVC++ 2003 Pro. Compile your Source SDK code without having to buy MSVC++. See this article for information on how. This process is... optimistic at best, though hypothetically, you should be able to get it to work.
Homepage



That about wraps it up. If you know of any other utilities that specifically help out with Source modding, post them in the comments of this page. A special thanks to all of these wonderful authors for producing tools for the benefit of the community, and thanks to Anarki for hosting the torrents at his site, Torrent Strike.
