ZHLT 3.0 Final Released !

Home » Content
Submitted by Andy on Mon, 2005-02-07 02:18.

Great news for those of you still designing levels for Half-Life 1 and also good news for those of you designing in Source, as Adam say's, there will be a new CST version out in the next few weeks.
Full details available below..

------ ------- ------------- ---------- ------ ------- ----------
From Adam Mckern:
So, here we go, the final instalment of ZHLT 3.0, and possibly the last version to be made.

I would like to thank ts2do, at TWL, for the help with the bug, which has been halting Final for about 3 Months now.

So yeah, ZHLT 3.0 Final has been released today, and is the last instalment of the ZHLT Tool set, that was started By Zoner, back in 1998.

The final version fixes the bugs from 3.0 Beta 4, and also speeds the code up to be more then 23% Faster then 2.5.3, on an average sized map (Table data collected by David Brophy)

Zoners 2.5.3 30 Final Difference
CSG 44.94 19.16 +57%
BSP 8.81 8.21 +7%
VIS 2m 36s 1m 41s +35%
RAD 5m 02s 5m 34s -11%
------ ------- ------------- ----------
Total 8m 32sec 7m 42s +10%

The reason why RAD is slower is because the memory leek has been fixed, and also the shadow code is more brutal in checking its self against the world.

So where do you download the new tools?

You can start by looking at www.zhlt.tk you can also download the three flavours of files direct from:

RAR
(884 KB)

ZIP
(1.04 MB)

EXE
(1.14 MB)

A Complete change log can be found here » http://downloads.ammahls.com/zhlt/changelog.txt

Now that ZHLT is now a completed project, expect to find a new Version of CST out with in the next few weeks with some radical changes.

Adam "amckern" McKern, and the ZHLT 3.0 Team
