KINGPIN

COMMON KINGPIN FAQ - by [DS]Method
Updates contact RUST


Preface:

Here's a bunch of KP frequently asked questions. Including mapping, gameplay and common questions.


MAPPING

When I try to compile a map, the DOS screen flashes fast and closes.
To compile your map you need to copy kpbsp.exe, kprad.exe and kpvis.exe into Kingpin/main/maps folder. You can find those files in the Kingpin/kprad folder.

When I set texture light flag in kprad's Surface Inspector, no matter what the color of texture light is always white.
When Xatrix was working on KP, they didn't have time to update ArgRad, and since KP based on Quake2 engine, it used wal files. After finishing KP, Xatrix recompiled textures to tga format, but forgot about wal lighting. You need to search the net for Kingpin wal textures.

I made Deathmatch/Bagman map and when I try to load it, it loads in SinglePlayer mode.
To load DM/BM map, you need to type in console "deathmatch 1", or start game with shortcut parameter "+deathmatch 1".

If I want to check the map design, do I have to wait all the rad and vis time?
You can pick "Bsp | bsp_only (novis/norad)" in kprad toolbar. It'll take less than a minute to compile, but you won't see lights and shadows in your map, it'll be fullbright.

If I created entity, is it possible to undo it?
Select your entity and pick "Selection | Ungroup entity" in kprad toolbar.

Is it possible to preview how func_door, func_train, etc works in game?
Select your entity and press "Ctrl-A", it won't work for specific KP entities, only Quake2 entities.

When I hire guys in my map they can't follow me.
You need to have navdata file in navdata folder. Load your map and type in console "nav_debug 1" and "nav_dynamic 1", now run everywhere you want your hired guys to follow. When you're done quit the game and navdata file will save itself in navdata folder.

GAMEPLAY

When I shoot at func_explosive or blow up characters, my KP crashes.
You need to install the latest KP patch - v1.21.

How can you zoom in KP?
You need to use "fov" - field of view. fov 90 is the default fov. fov 180 is the highest fov. The smaller the number the more you zoom. Some players like to play with fov 120 cause you can see alot more than with the default fov.

What's doublejump?
Doublejump is a kind of jump. To do it, you need to find something small (less then 64 * 64 brush), then run toward it and before touching it, press jump twice fast. If you did it correctly you should jump higher than normal jump.

What's bunnyhopping?
Bunnyhopping is another kind of jump. To do it, you should press right strafe and turn your mouse 45 degrees to the right side, then press jump, keep jumping and cirle, you'll gain speed. You can use right or left strafe to bh or you can use them in combination. Don't press forward, the bunnyhop won't work, use only left or right strafe.

Is there a weapon against bazooka?
Flamethrower is a great weapon against bazooka, flame kills rockets, so if you aim at the enemy's bazooka he'll blow himself up.

COMMON

What voices Cypress Hill did in KP?
They did voices of bartender, Momo and Jesus.

What things didn't make into final version of KP?
There was different kinds of mods for weapons. Three kinds of armor, chemical, flamable and bulletproof. Bumpmapping and drivable car, etc.

Was there any movies, which inspired KP?
Some of KP dialog was inspired by Pulp Fiction and Big Lebowski.
