ammo_357

Spawns ammo for the .357 Magnum.

Properties

Name - Provide a name to use as a trigger for this entity.

Pitch Roll Yaw - Sets the pitch (up / down), yaw (left / right) and roll (bank) respectively. The compass in WorldCraft / Hammer corresponds to Yaw. The settings are not always (or not all) used.

Target - When the entity is activated or picked up, the Target named here is triggered.

Delay before trigger - Usually the time in seconds before an entity should trigger its target (after being triggered itself).

KillTarget - When If you pick the ammo up or use it's name (trigger it), the entity named here will be removed from the game completely.

Flag Properties

(2084) Not in Deathmatch - If this flag is set on, the entity will not appear in deathmatch games.

Weaponbox

Creates a canister full of specific ammo.

Notes:

To stock the weaponbox, turn SmartEdit off, click Add, and type one of the values below as the Key:

    357
    9mm
    ARgrenades
    bolts
    buckshot
    Hand Grenade
    rockets
    Satchel Charge
    Trip Mine
    uranium

The amount of ammo you want to add to the box, goes in Value. I believe there are limits, but haven't tested it
