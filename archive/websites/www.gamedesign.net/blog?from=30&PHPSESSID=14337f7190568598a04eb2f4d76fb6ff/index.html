Visual Studio .NET 2003

Submitted by yesukai on Mon, 2005-03-28 01:16.

My friend at the tech college came through for me again, this time with a cheap copy of Visual Studio .NET 2003. I love it so far, compiles the source code without any errors or any damn workarounds. Just pure, clean "lets get down to business" coding.

What this mean for you guys is the same outcome as me learning XSI and the new HL2 ents: As I learn something, it get put into a tutorial. The more I learn, the more tutorials Rust gets. :)

Im going to start with something easy for my first mod; some guys in IRC were nostalgic about Runes for Quake, so I figured, hey, that sounds easy. So, I'll be making a ruins mod for HL2MP, and, even better, I'm going to heavily document it so that I can make tutorials about my experience. An entire mod creation, all down so you can follow along (after im done of course, dont want to lead people down the wrong path when I screw up. :) ). But that should make an interesting load of articles for HL2->Coding.
