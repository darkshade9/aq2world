How to create fixed size trigger?

TRIGGER_RELAY

This fixed size trigger cannot be touched, it can only be
fired by other events.

Example use: create func_button and open Entity inspector.

Key angle
Value 0

 Value determines the opening direction.

Key lip
Value 5

 Value is the override the default 4 pixel lip remaining at
end of move.

Key sounds
Value 4

 Value is the sound number.

Key target
Value t86

 Create trigger_relay and open Entity inspector.

Key targetname
Value t86

 Value is the trigger_relay target ID.

Key target
Value t88

 Value is the targetname of target_speaker.

Create target_speaker and open Entity inspector.

Key spawnflags
Value 4

 Value is the flag reliable.

Key targetname
Value t88

 Value is the target_speaker target ID.

Key noise
Value world/pawnbuzz

 Value is the path for the sound.

You can also add delay to trigger_relay.
