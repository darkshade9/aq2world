How to group characters in gang?

LOCALTEAM

Localteam used to group AI in gangs. Like gang, Sewer Rats
in Skidrow. They're your enemies, but sometimes they warn
you like "What the f*ck you lookin' at?". If you kill
gang member they start blasting you without saying anything.

 Example use: select all characters, which you want to be
in the gang and open Entity inspector.

Key localteam
Value sewer_rats

 Value is the name of the gang. You can change it on whatever you want.
