How to unlock a targeted door?

TRIGGER_UNLOCK

Player will unlock a targeted door when this brush is
touched

Can be triggered also!
door's "key" should equal -1
"target" target ID to match door

 Example use: create func_door and open Entity inspector.

Key angle
Value -1

 Value determines the opening direction.
-1 opens up, -2 opens down.

Key key
Value -1

 Value is the key number.
If key is -1, that means door is closed.

Key sounds
Value 6

 Value is sound number.

Key targetname
Value t120

 Value is the func_door target ID.

Key wait
Value 6

 Value is the time to wait before returning (3 default, -1 =
never return)

Create trigger_unlock and open Eintity inspector.

Key target
Value t120

 Value is the targetname of func_door.

Place trigger_unlock behind a func_door.
