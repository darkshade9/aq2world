How to divide the world into areas?

FUNC_AREAPORTAL

This is a non-visible object that divides the world into
areas that are seperated when this portal is not activated.
Usually enclosed in the middle of a door.

 Example use: select func_door_rotating and open
Entity inspector. Find fields with "key" and "value".

Key target
Value t94

 Value is the targetname of func_areaportal.

Select func_areportal and open Entity inspector.

Key targetname
Value t94

 Targetname of func_areaportal.

Key style
Value 1

 Style used as areaportal number.

Place func_areaportal in the middle of a door
(func_door_rotating).
