How to make character head towards path_corner_cast?

PATH_CORNER_CAST

Character will head towards the specified path_corner_cast.
Pathtarget: gets used when an entity that has this
path_corner targeted touches it. Could be used to
trigger a button, or a character. (see pathtarget tutorial)
Combattarget: when the cast reaches this marker, they'll
take on this combattarget. (see combattarget tutorial)

 Example use: Select character entity and open
Entity inspector.

Key target
Value t24

 Value is the targetname of path_corner_cast.

Select path_corner_cast and open Entity inspector.

Key targetname
Value t24

 Value is the targetname of path_corner_cast.

Key target
Value t25

 Value is the targetname of another path_corner_cast.

Select another path_corner_cast and open Entity
inspector.

Key targetname
Value t25

 Value is the targetname of path_corner_cast.

If you want to make a loop, add this to the last
path_corner.

Key target
Value t24

 After character touches last path_corner, he'll go to the
first path_corner.
