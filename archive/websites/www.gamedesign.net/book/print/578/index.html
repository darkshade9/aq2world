Preface:

This tutorial illustrates how to make transparent shaders.

Step 1

Perhaps you've tried to make something that looks like this:


But you haven't found the right texture for it? Well, why not make your own. First, we need to do the texture, i'll use Photoshop. Here's what my texture looks like:


Now, save this file as a 24- or 32bit TGA file. While we are in Photoshop, do a selection at the areas that should be visable. In this case you should select the "Brown" areas.


(Note: this image is enlarged)

Now, save the selection (under the "Select" menu in Photoshop), and save the picture. You have just made the base for your transparent Texture, now it's just the Shader writing left.

Step 2

Now, open Notepad and type in this:

textures/mytextures/transparenttexture
{
    surfaceparm trans
    surfaceparm alphashadow
    surfaceparm nomarks
    cull none
    nopicmip
    {
        textures/mytextures/transparenttexture.tga
        blendFunc GL_ONE GL_ZERO
        alphaFunc GE128
        depthWrite
        rgbGen identity
    }
    {
        map $lightmap
        rgbGen identity
        blendFunc GL_DST_COLOR GL_ZERO
        depthFunc equal
    }
}

Note: I've called my texture "transparenttexture.tga", and it's located in textures/mytextures.

There will be no bullet marks on this texture, cause we have the "surfaceparm nomarks" on it. Also, it will not give any shadows. The rest fixes so the texture will look right compared to the lightning and so on. Now, save this file in the scripts directory, i'll save mine as "my.shader" (remember to save it as *.shader), then add it to the shaderlist.txt file.

Here we go, load up qeradiant, and start using your new texture!
