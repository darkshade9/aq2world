How to create character's combattarget?

COMBATTARGET

When the cast reaches this marker, they'll take on this combattarget.

Example use: Select a character and open Entity inspector.

Key combattarget
Value combat1

 Value is the targetname of path_corner_cast.

Select path_corner_cast and open Entity inspector.

Key targetname
Value combat1

 Value is the targetname of path_corner_cast.

Place it away from the character.
