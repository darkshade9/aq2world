How to set character's accuracy?

ACC

Acc is a weapon accuracy. It used only with characters
(cast_bitch, cast_punk, etc.). There is 5 kinds of accuracy
for AI.

ACC_GREEN 0
ACC_POOR 1
ACC_NORMAL 2
ACC_SEASONED 3
ACC_VETERAN 4
ACC_ELITE 5

 Example use: select a character in KPRadiant and hit "n".
You'll open Entity inspector. Find fields with "key"
and "value".

Key acc
Value 3

 This AI will have accuracy of ACC_SEASONED.
Accuracy also used with melee weapons.
