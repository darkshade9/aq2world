How to create gang war?

AI_EVENT_HOSTILE

Character touching this brush will become a
hostile enemy to all other characters in LOS, that have
the same "cast_group" as the brush.

 Example use: select 2 characters and open Entity inspector.

Key cast_group
Value 153

 Value is the number of their cast_group.

Key localteam
Value postbattery1

 Value is the name of their localteam.

Key targetname
Value 119

 Value is the targetname of the character.

Select another 2 cast_ entities and open Entity inspector.

Key cast_group
Value 152

 Value is the number of their cast_group.

Key localteam
Value postbattery2

 Value is the name of their localteam.

Key targetname
Value 119

 Value is the targetname of the character.

Create ai_event_hostile brush with x: 64 y: 64 z: 32 and
place it on the ground. Select it and open Entity
inspector.

Key cast_group
Value 152

 Place it inside character with cast_group 153.
That means that this cacharacter and members of his
localteam is enemy of cast_group 152.

Create another ai_event_hostile brush with X: 64 y: 64 z:
32 and place it on the ground. Select it and open
Entity inspector.

Key cast_group
Value 153

 Place it inside character with cast_group 152. That means
that this character and members of his localteam is
enemy of cast_group 153.

Create trigger_once brush with x: 32 y: 288 z: 128 (or x:
128 y: 16 z: 128)

Key target
Value t119

 Value is the target of character.

Place this brush away from characters.
So when player trigger trigger_once, characters will start
blasting.

To see this tutorial load sr2.bsp, get bike battery and
watch Rats and Scorps waste each other.
