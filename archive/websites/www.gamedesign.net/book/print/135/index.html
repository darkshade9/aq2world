How to set character's skins?

ART_SKINS

Art_skins used as a skin for a model. Art_skins hold 3 -
digit skins for head, torso and legs.

 Example use: select a character and open Entity inspector.

Key art_skins
Value 041 026 010

 Value is skin number. To find all skins go to
main\models\actors.

Look in thug, runt and bitch folder.
