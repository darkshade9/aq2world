How to create pathtarget?

PATHTARGET

Pathtarget used with path_corner. Any entity which will
follow this path_corner will activate entity with targetname
of pathtarget value.

 Example use: select path_corner and open Entity inspector.

Key pathtarget
Value opengate

 Value is targetname of entity which will activate after any
entity will follow this path_corner. You can enter any name
for pathtarget value.
