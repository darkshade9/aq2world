How to create characters with different kinds of head?

HEAD

Head used with characters for different kinds of head.

Bitch's head:

Bald 1
Pony tail 2
Default 0 or anything else

Punk's head:

Bald 1
Ski 2
Weld 3
Default 0 or anything else

 Example use: select a character and open Entity inspector.

Key head
Value 1

 This character will spawn bald.
