How to set character's currentcash?

CURRENTCASH

Currentcash is how much money entity has. You can put any
value for money. Used with characters.

 Example use: select a character and open Entity inspector.

Key currentcash
Value 8

 It also used with cast_group 1 (friendly AI). Select entity
with cast_group 1 and open Entity inspector.

Key currentcash
Value -10

 You need to pay $10 to hire him.
