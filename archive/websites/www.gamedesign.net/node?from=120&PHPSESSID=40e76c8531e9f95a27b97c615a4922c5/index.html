December is a good month for game dev jobs.

Submitted by skinmaster on Mon, 2004-12-27 17:20.

I'm the type of guy who's constantly scouring the web for references and such. One of the thing I look for is game dev jobs or game industry jobs. That's site's listing positions offered by game dev companies.
