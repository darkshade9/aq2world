Q3MAP2 - Version 2.5.14 released

Submitted by Miniwood on Sat, 2004-04-10 22:49.
Ydnar announces the new version of Q3MAP2, the best compiler for Q3A and games based on the Q3A engine. New features include: - New: -dirty mode courtesy of RaP7oR (dirtmapping/ambient occlusion) - New: q3map_skyLight code using more uniform angular distribution about the sky hemisphere - New: q3map_alphaMod set N.N - New: q3map_alphaMod scale N.N - New: q3map_alphaMod volume - applies all other q3map_alphaMod directives to each vertex inside a brush textured with this shader, allowing large faded scrolling fire shaders, waterfalls, marquees, explicit dotProduct terrain blending control, etc.

The Wadfather is back

Submitted by Miniwood on Sat, 2004-04-10 22:41.
The site that calls itself 'The Ultimate Collection of Public Domain Graphics for Game Developers' has relaunched with a new look and easier to use interface. The Wadfather is a great source for graphic files for Half-Life, Q3A and Unreal and currently boasts: * 475 environment maps * 184 texture collections with 17902 textures * 62 desktop wallpapers The latest update is a new texture set by evillair based on the classic Quake 2 textures which you can download in Q3A, Half-Life & Unreal flavours.
