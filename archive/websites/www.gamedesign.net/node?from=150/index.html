Halo Custom Edition & Halo Editing Kit Released

Submitted by Miniwood on Thu, 2004-05-06 09:28.

The Halo Custom Edition and Editing Kit allowing users to add their own content has been released by Gearbox Software with little fanfare and no announcement on either the Gearbox or Bungie websites.

The Custom Edition is a standalone multiplayer add-on for the Halo: Combat Evolved PC game which adds support for user-created content. You'll need a valid CD KEY from the original game to run it.

The Editing Kit comprises all the tools you need to create your own maps and content for the game.

The add-on and the editing kit is available as one 170Mb download from 3DDownloads.

MOD development tutorials

Submitted by G.A.R.D. on Wed, 2004-05-05 01:53.

I don't want to advertise ... but I followed the "SOLDNER: Secret War" advert link to their home page. I discovered 6 tutorials documenting the steps the developers went through to create the game. It was a very good read, well written, good presentation, and clearly describes each step in MOD development. The tutorials do lack a detailed technical description of the issues faced at every step. However, it does provide a realistic overview of the task you have to accomplish when making a MOD. Too often people pop onto the RUST forums with a so-called great MOD idea, and want someone to help model the weapons. There must be more weapons for failed models than any other part of the game. The "SOLDNER: Secret War" development tutorials puts MOD developments steps in the right order (you don't make the guns first!). So it is a good read if you are planning to make your own game.
