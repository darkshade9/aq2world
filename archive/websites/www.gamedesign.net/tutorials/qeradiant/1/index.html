Getting Ready for Quake3:Arena Editing


Tutorial 1 : Downloading, Installing, and Setting Up


This is the beginning of a series of tutorials aimed at getting people ready for Quake3:Arena Editing. Since Q3radiant will be released on the CD with the game, we should be seeing some level editing action very shortly afterwards.

Q3Radiant will be a updated and revised version of the current Quake2 editor QERadiant. So naturally being familiar with QERadiant will allow the transition to Q3Radiant that much easier.

These tutorials will be aimed at beginners and people who just wanting to convert to QERadiant from another editor.

Let's begin.

Downloading

Naturally the first thing you are going to want to do is download the QERadiant editor and other misc. files. You can download the newest version of the software here. Make sure you get any patches also. With this tutorial I am using 1.47

I am assuming all of you know how to install software. QERadiant is no different. Click on the install icon, select the directory to install the program in, etc, etc. Unlike other editors, QERadiant can be installed anywhere and on any drive. It doesn't have to be on the same drive as the game.

After installation is complete you should see a nice looking new QERadiant icon on your desktop.

Double click on it.

Setup

After launching the program you will be confronted with a big window of preferences. Below is an image of Fishman's Recommended Preferences:



Choosing the Right Rendering Mode

One very important thing that you will have to change is what rendering mode you will use in QERadiant. There are 3 choices: Use SGI OpenGL, Buggy ICD, or nothing. By default no rendering mode is chosen. If you have a Nvidia TNT, or another type of full OpenGL accelerator then stay with no rendering mode chosen. QERadiant will detect your OpenGL drivers and use those.

If you do not have an OpenGL accelerator then choose either Buggy ICD or Use SGI OpenGL. Either one will work. Though I would test both and see which is faster. If you choose Use SGI OpenGL, make sure you have the SGI OpenGL drivers installed. The drivers come with Windows 98, Windows NT4, and Windows 95 OSR2. So the only case I can see where you would have to download the drivers is if you are running the old version of Windows 95. If you happen to be one of these people you can download the OpenGL drivers here.


3 button or 2 button?

Only choose the three button mouse if you have the third button of your mouse set to THIRD BUTTON. you get more advanced functionality if you do set the third button though. I recommend it.


Setting the GAMEPATH

in the gamepath section, enter the path to your Quake2.exe file, and set the use pak file path to your quake2 pak0.pak file. if these paths aren't correct you will have some trouble with the next tutorial.

Don't worry if you really don't know what all these options do. we will go over them in due time.


The Rest of Setup



After clicking OK, you will be asked if you want to create a new project:



Click YES


Now you will be asked for a name of the new project:



type in rusttuts or whatever for the project name and click OK


Now the program will load and you should see a screen that looks similar to this:(click to enlarge)



If your screen looks like the one above then congratulations! you have successfully setup and installed QERadiant and completed the first QERadiant tutorial here at Rust.

What you are seeing is the default map that QERadiant loads. In the next tutorial you will get familiar with the QERadiant Graphical User Interface.

If you had problems with this tutorial or found an error or just a better way I could of explained a concept then please post on the QERadiant Forum.
