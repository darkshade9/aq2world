Getting Ready for Quake3:Arena Editing


Tutorial 3 : Loading, Searching for, and Selecting Textures


In this tutorial you will begin the process of making your first "hello world" map. We will concentrate on loading, searching for, and selecting textures.

First thing we need to do is start a new map from scratch. So goto FILE, NEW MAP on the menu. After this your screen you should look like the once below:



Selecting the Rendering Quality

First think you are going to want to do is select a texture/3d window rendering mode. This is done by clicking on the Textures menu and then the Render Quality submenu. The image below should illustrate it for the lost ones:



There are 8 different rendering types for the 3D View:

  • Wireframe
  • Flat Shade
  • Nearest
  • Nearest Mipmap
  • Linear
  • Bilinear
  • Bilinear Mipmap
  • Trilinear

These are basically ordered in less to most complexity. If you do not have a OpenGL accelerator like a Nvidia TNT then choose either Nearest, or Nearest Mipmap. If you do have a graphics card with a full OpenGL driver like a Nvidia TNT then I recommend choosing Bilinear, or Bilinear Mipmap. If you have a really really slow computer then you might want to choose Wireframe or Flat Shade, but I don't recommend it.


Loading Textures

The textures in the texture browser are just the ones that were used in the default map before. We are definitely going to need some more textures to make a cool map, eh? Goto TEXTURES on the menu, and then select E1U2. For those of you lost, an image is included below to help:



A dialog box will come up that should look sorta like the one below



Make sure "Load all Textures" is selected and click "OK". After clicking ok, you should see in the console window at the bottom, all the textures being loaded from the PAK file. After it is done you should see the textures from E1U2 in the texture window all ready to use.


Finding and Selecting a Particular Texture

Now we need to select a texture to use for a first room. To make this tutorial simple let's just make our map a one texture map, and let's use the texture metal1_3. To navigate through the textures you can either use the scroll bar or hold down the RIGHT MOUSE BUTTON and drag the texture list up or down. Who want's to spend all the time looking for textures though. We know the name, lets search for it! QERadiant has a handy filter/search tool to find a texture that your looking for.

in the text input box above the texture window type metal. As you type you should see some textures disappear with each letter. after finishing typing metal, you are left with all the textures that begin with the word metal. Below is a screenshot for those of you who are lost:



It should be easy to find the texture metal1_3 now, of course you could just type the whole texture name into the filter box if you want to. DOUBLE CLICK on the texture to select it.


Changing the Texture Window Scale

Another cool option that is found under the Texture menu is the option to scale down or up the size of the textures in the texture window. try it out for yourself. Goto to the Texture Window Scale submenu under the Texture menu and choose the scale that you like the best. For the clueless the screenshot below should help:



Personally I like the default value of 100%
Well that sure was short and sweet. Of course there are more texture options, but we shall get to those in due time. In the next tutorial you will learn how to create the basic architecture for your first map.
