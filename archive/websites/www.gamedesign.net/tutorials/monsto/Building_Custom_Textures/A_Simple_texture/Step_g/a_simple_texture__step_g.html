g. Adding the grunge

Last thing then you can call this baby done. We're going to add a bit of concrete-scratch in order to give that center area the `worked concrete' look.

  1. Open DSCN0127[1].jpg into a new picture.
  2. Select and copy a section the size of our texture: 256 x 512. It's a big image so that shouldn't be too tuff.
  3. Back in wall_1, add a layer and paste the selection. Call it ``grunge" then hide the layer.
  4. Get the Wand tool again, select inside the top bevel of the layer ``inner bevel" and if you can, add-select the bottom section. (if not, it's 2 steps.)
  5. Make visible and select the ``grunge" layer to edit. you should have the selection over the pitted image.
  6. Invert the selection so that everthing BUT the bevel centers are selected. The marquee will look like an 8.
  7. Delete the selection (press the delete key and lets see what happens).

You will see the selection marquee and the ``grunge" layer image. Deselect to make sure nothing is selected.

At this point you can mess with the layers Blend to make it work. Overlay with some transparency (~50%) and Soft Light seem to work best. Luminosity with some transparency keeps the concrete color intact, but in doing so makes it look flat without the small changes in color that Soft Light leaves in.
