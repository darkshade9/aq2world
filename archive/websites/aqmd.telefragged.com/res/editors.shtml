Editors
Here are some of the most common applications used to create items on this website. Do their makers a favor and buy them (if applicable). Don't be a software pirate. Or, if not for any other reason, don't be a cheap-ass, parasitic bastard!

BSP Files

Program: QERadient
Location/Filename: http://www.qeradiant.com/
Developer: raduffy@cyberramp.net
Description: Another freeware map editor. This program is based on the source code for QE4, the map editor that ID software used to create Quake2. Handles Quake, Quake2, Heretic2, Half-Life, and Sin.

Program: Qoole 99
Location/Filename: http://qoole.gamedesign.net
Developer: Lithium Software
Description: A freeware map editor. Qoole v0.99 is the final version which is available. Handles Quake, Quake2, Heretic2, Half-Life, and Sin.
Qoole has gone Open-Source and is licensed under the the terms of the GPL (General Public License). Visit the Qoole Source Repository and see what's going on.

Program: ArghRad 2.0
Location/Filename: http://www.planetquake.com/arghrad/
Developer: Tim "Argh!" Wright
Description:An enhanced version of qrad for compiling bsp format files.

PAK Files

Program: PakExplorer v1.2
Location/Filename: http://quakestuff.telefragged.com/
Developer: Quakestuff
Description: THE tool for viewing PAK files from AQ2.

Program: QPed II v2.11
Location/Filename: http://www.planetquake.com/qped/
Developer: Pete Callaway
Description: Another very good PAK editor. I prefer it myself, though I was a "hardcore" user of PakExplorer.

WAL Files

Program: Wally Texture Editor
Location/Filename:http://www.telefragged.com/wally/
Developers: Ty Matthews & Neal White III
Description: Awesome tool for viewing and modifying WAL files located in Action Quake 2's PAK files.

WAV Files

Program:Acid WAV
Location/Filename:http://www.polyhedric.com/software/acid/download.html
Developer: Polyhedric Software
Description: Mix, modify, and listen to WAV files.

Program: Goldwave
Location/Filename: http://www.goldwave.com/release.html
Developer: Goldwave
Description: Awesome tool for mixing, modifying, and listening to WAV files. This version is a fully operational demo.

BMP, GIF, JPG Files

Program: Paint Shop Pro
Location/Filename:http://www.jasc.com/download.asp
Developer: JASC Software
Description: Poor man's version of Photoshop - layers, anti-aliasing, bezier curves, everything ya need for awesome graphics on a budget. Photoshop plug-ins are also compatible with this program. Hyperlink is for the evaluation version.

Program: Photoshop
Location/Filename:http://www.adobe.com/products/tryadobe/main.jhtml#product=39
Developer: Adobe
Description: Simply the state of the art for 2D graphics tools - if you have the money, this program can graphically do whatever you want to do. Hundreds of plug-ins for those cool special effects. Hyperlink is for 30 day demo version.
