Map Search
You can search for maps in two different ways:
  1. By entering some text and comparing against the zip file name, map name, map title, author or any combination of these fields.
  2. By selecting the types of maps you would like to see, including minimum and maximum players and release date.
Search for:
In: Zip Name   Map Name   Map Title   Author
 
Only show maps for:
DM   TP
Action Espionage (AE)   Breakable Glass (BG)   LAN Only (LAN)   CGF
Released: Before   After     (MM/DD/YYYY)
The two search options are mutually exclusive, meaning that if you decide to do a text search then the map type is not checked and visa-versa, if you select the Type Search button then no text comparisons are done.
