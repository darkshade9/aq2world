New Maps!!
Posted Tuesday, October 28, 2003 @ 4:24PM [EST] by GreyDeath

Well it's true! New maps hot off the compiler!

First we have Lundvall's Dive! It is a lovely map with a nice waterfall and some good texture work. This is also a first submission for Lundvall.

Next up, Bobert's Miner 69'er map! Mine is a mining colony smack dab in the middle of a rocky desert. Some nice terrain work done on the outside, and an impressive texturing job on the inside. This is definately worth a look at.

3rd, we have Lundvall's Storm. Storm is a winter city map with a nice Christmassy theme.

And last, but not least, Wizs bye, yet again, Lundvall! Wizs is very similar in style to Storm. However, this one is without snow and has a different layout.

All 4 maps are definitely worth checking out. Also, I have added a link to a new tutorial dealing with hint brushes. This is probably the best tutorial I have ever read that talks about hint brushes. Instead of only talking about them from a top-down perspective, it takes into account all 3 axis. This is a must read for all mappers out there. Check it out here!

Lastly, if you have a map waiting to be submitted, please let us know. You can tell us about it and provide a link to the map at the AQMD Forums.

GreyDeath out.
