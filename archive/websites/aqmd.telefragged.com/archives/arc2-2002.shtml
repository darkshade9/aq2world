More updates!
Posted Sunday, March 3, 2002 @ 6:19PM [EST] by MP*Idle

Yap, yap!
Updated the map submission requirements page and the Sounds page.

Uhm, yes... so, I made two sound packs for our mappers,
one has some broad selection of animal sounds,
and the other one has several ambient sounds.

Then, about the map submission requirements update:

When you're zipping your map,
make sure the readme file (the text file) is named like the map,
for example: My map is named "sillymap.bsp", then the readme file would be named "sillymap.txt".

Also, the mapname.txt file should extract to a directory called "mapdocs".

So, it'd be something like this:


  • .bsp files go in c:\quake2\action\maps
  • .wal files go in c:\quake2\action\textures\myname
  • .wav files go in c:\quake2\action\sound\myname
  • .pcx and .tga files go in c:\quake2\action\env
  • .txt files go in c:\quake2\action\mapdocs


Also, here is a template for your map readme files: View Template
I suggest you save a copy to your hard drive and use it when submitting your maps, it'd make life so much easier for us when releasing the map.
