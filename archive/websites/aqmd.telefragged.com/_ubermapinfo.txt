
  All the Official Action Quake 2 maps as of 7/30/2000.  These were 
  all gathered from the Action Quake Map Depot.  There are 181 maps 
  total.  You can check for more at:  http://aqmd.telefragged.com/


  Please read the following files:

  maps/docs/_mapfix.txt
  maps/docs/_maplist.txt
  uber-nix/readme.txt    (*nix users READ THIS, Linux, FreeBSD, etc.)


  Layout:

  env/       (environment images)
  maps/docs/ (map documentation and info files)
  sound/     (sound files for maps, models, etc.)
  textures/  (textures for maps)
  uber-nix/  (*nix users NEED TO RUN THIS SCRIPT FIRST)


  Files:

  mapinfo.txt (you're reading this)
  pak2.pak    (431 MB, contains .bsp's, env, sounds, textures, etc.)
  pak3.pak    (45 MB, contains only additional textures)


  Total size: 

  470 MB uncompressed
  180 MB compressed in ZIP format

  RAR or ACE would compress better but I don't know if 
  *nix users can uncompress these kind of files.


  Installation:

  Simply uncompress this ZIP file into your Quake2 directory.
  If there are any PAK files named "pak2.pak" or "pak3.pak", then you 
  better copy them to a different name or another place lest they get 
  overwritten.  This map package replaces ALL other AQMD map packages 
  so you can delete them--which will save space and redundancy too.

  *nix users, Linux, FreeBSD, etc., NEED to follow the directions found 
  in the "uber-nix/" directory IMMEDIATELY AFTER UNCOMPRESSING THIS ZIP.
